Instructions:
************
-Navigate to your StarCraft folder.
-Find a file called patch_rt.mpq
-Back this file up somewhere safe, as you will need to put it in your StarCraft folder again
when you want to play regular StarCraft.
-Place the patch_rt.mpq file that came in this zip file in your
StarCraft folder.

Note:
As of version 1.4, there are now actually two mpq's, only one of which is to be used at any
given time.  The "Insane Mode" mpq represents what used to be the only "mode."  Due to the
extremely hardcore awesomeness of this mpq, many players actually found it too difficult.
Hence, the birth of "Normal Mode."  This mode should be easier, but still present a nice
challenge.

-OK, you are ready to play!  Have fun playing against some AI that can actually present a
challenge!
-As stated previously, to switch from the mod to normal StarCraft, just replace the mod's
patch_rt.mpq file with the one that came with StarCraft.


ATTENTION:
----------
-This mod is only compatible with StarCraft: Brood War v1.16.1.
If you have an older version, get on Battle.net and get the latest version!
-Also, a note on patching the game: If you connect to Battle.net and it starts updating,
cancel out, close StarCraft, and make sure that the patch_rt.mpq file is the real StarCraft
one, not the one for the mod.  Then start StarCraft up, patch as normal, then download the
new version of the mod(there will usually be one released pretty soon after a new StarCraft
patch).
